AWESOME CSS ANIMATIONS

  • Today I'll be describing to you the creativity make possible to webpages through CSS by transition and the @keyframe rule.
  • What is CSS Animation
  • CSS animation is a model of CSS that defines how to animate the values of CSS properties over time, using keyframes, and the behavior of these keyframes can be controlled by specifying their duration, number of repetition, and how they repeat.
  • A little history lesson. Animation was first introduced by Webkit in 2007 and slowly over the years web browsers such as Firefox and Chrome began to support it.
  • However, CSS3 animation became incresingly popular over traditional script-driven animation due to its ease of use, readibility and scalability.
  • While transition allows users to trigger animations by means of Javascript, @keyframe allows repeated animations non-dependent on user.
