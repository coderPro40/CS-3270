JavaScript References

The javascript commands I know so far are as follows:

Control Statements

  • if statement: this iterates once if condition is met.
  • While loop: this iterates continuous until condition is met, if no condition is given or if it's always true, will become infinite loop.
  • For loop: this consist of 3 sections, the start section, the stop section and the iteration section to control the loop; note that if none is provided, we get an infinite loop.

Arithemetic Operators

  • + operator: this is used to add two variables of type integer.
  • - operator: this is used to subtract two variables of type integer; can also serve as unary operator to conver from positive to negative.
  • * operator: this is used to multiply two variables of type integer.
  • / operator: this is used to divide two variables of type integer.
  • % operator: this is used to get the remainder from division of variable of type integer.
  • ++ operator: this is used to increment a variable of type integer.
  • -- operator: this is used to decrement a variable of type integer.

Assignment Operators

  • = operator: this is used to test whether two variables of same data type are equal in value.
  • += operator: this is used to increase or append another value to current value of same datatype.
  • *= operator: this is used to multiply another value to current value of integer datatype.
  • /= operator: this is used to divide another value to current value of integer datatype.
  • %= operator: this is used to get the remainder from division of another value to current value of type integer.
  • &= operator: this is used to perform a bitwise and on two values of type integer.
  • |= operator: this is used to perform a bitwise or on two values of type integer.

Functions

  • These can be defined using the function keyword.
  • These can also consist of parameters such as work(parameter 1, parameter 2) depending on user choice.
  • Parameters passed in serves as arguments for the function.
  • return keyword is used to terminate the function and return control to the user; used to return result to user.

Strings

  • These used for storing texts and are defined using the var keyword.
  • Strings consists of properties such as length and this is retrieved using the .length keyword.
  • Substrings can also be extracted from strings using the slice(), substring(), and substr() keywords.
  • Can also search for text within substring using the indexOf() and search() keywords.

There are many more Javascript commands omitted from this document, but for the sake of being brief these are the first few that come to mind.
